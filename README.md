# 🛒 Shopping List
 
A simple terminal program to never forget what to buy at the market again!
 
## 💡 About
 
**Problem solved:** *"I get to the market and forget what I needed to buy!"*
 
This is one of my first Python projects, made to practice logic, terminal interaction and code organization.
 
## ⚙️ Features
 
- ✅ Add item to the list
- 📋 Show all items
- ❌ Remove item *(coming soon)*
- 🚪 Exit
 
## 🚀 How to run
 
**1. Clone the repository**
```bash
git clone https://github.com/seu-usuario/shopping-list.git
```
 
**2. Enter the folder**
```bash
cd shopping-list
```
 
**3. Run the program**
```bash
python shoplist.py
```
 
> Requires [Python 3.10+](https://www.python.org/downloads/) — project uses `match/case` syntax.
 
## 🖥️ Example
 
```
1 - Add item
2 - Show list
3 - Remove item
4 - Exit
 
Choice: 1
Type your item: RICE
✅ RICE added!
 
Choice: 2
1. RICE
2. BEANS
 
Press enter to back:
```
